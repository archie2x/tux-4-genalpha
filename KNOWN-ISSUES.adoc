= Known issues

== Out of scope

[cols="2,3"]
|===
| _LAN multiplayer (tuxmath)_
| SDL_net is not part of SDL3. The networking subsystem is stubbed in
  `tuxmath/src/sdl3_compat.c`; "Run Server" / "Join Game" / "Stop
  Server" entries are no-ops. Single-player and local turn-based
  multiplayer work normally.

| _tuxmathadmin tool_
| The CLI helper for managing per-student progress isn't part of the
  build.

| _SDL2 / SDL1 builds_
| Source compiles only against SDL3.
|===

== tuxmath

[cols="2,3"]
|===
| _Factoroids ship rotation looks slightly hard at small zoom levels_
| Bilinear rotozoom shim in `sdl3_compat.c`. Pre-rendering, or moving
  to SDL_Renderer with a transform, would smooth it.

| _F10 fullscreen + macOS Spaces interaction_
| Alternating between F10 and the green traffic-light button rapidly
  may leave the menu bar non-auto-hiding until the next windowed →
  fullscreen cycle.

| _TTS announcer is mostly no-op_
| The text-to-speech path is stubbed (`T4K_Tts_say` etc.). No crash;
  no speech.
|===

== tuxtype

[cols="2,3"]
|===
| _Theme switching mid-session_
| Changing the active theme via Options doesn't relay out already-
  rendered panels until you bounce back to the title screen.
|===

== Reporting

File issues at https://github.com/archie2x/tux4kids. Include OS,
SDL3 version, and `-d` debug output.
