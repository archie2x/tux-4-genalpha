= tux4kids

Two educational SDL3 games and their shared support library.

[cols="1,3"]
|===
| `t4kcommon` | Shared library: menu engine, audio, image / font loaders,
                line wrap, theme machinery.
| `tuxtype`   | Tux Typing — fish cascade, comet zap, lessons, phrase
                practice, theme + word-list editor.
| `tuxmath`   | Tux of Math Command — comet zap with arithmetic, fleet
                missions, training academy, factoroids / fractoids.
|===

Submodules live on `a2x/sdl3_port` of `archie2x/t4kcommon`,
`archie2x/tuxtype`, `archie2x/tuxmath`.

== Quick start

[source,sh]
----
git clone --recursive git@github.com:archie2x/tux4kids.git
cd tux4kids
cmake --preset Debug
cmake --build _build/Debug -j
cmake --install _build/Debug --prefix _build/Debug/stage
./_build/Debug/stage/bin/tuxtype     # or .../bin/tuxmath
----

See link:BUILD.adoc[BUILD.adoc] for prerequisites and per-platform
notes.

== Platforms

* macOS (Apple Silicon, Intel)
* Linux (Debian 12+, Ubuntu, RHEL family)
* Windows (cross-compiled via mingw-w64)

== Dependencies

CMake's `find_package` is intercepted by a provider in
`cmake/modules/t4k_provider.cmake` that resolves each dep in order:

. In-tree target already in the build graph.
. System install (`brew`, `apt`, etc.).
. CPM fallback — fetches and builds at the version pinned in
  `cmake/package-lock.cmake`.

A missing system package therefore isn't fatal: the provider builds
the dep into `_build/<preset>/_deps/` on first configure.

== Layout

[source]
----
tux4kids/                 super-project, this repo
  CMakeLists.txt          orchestrates t4kcommon + tuxtype + tuxmath
  CMakePresets.json       Debug / Release / RelWithDebInfo / MinGW-Win64
  cmake/                  dep-provider, package-lock, toolchain files
  t4kcommon/              submodule (archie2x/t4kcommon @ a2x/sdl3_port)
  tuxtype/                submodule (archie2x/tuxtype   @ a2x/sdl3_port)
  tuxmath/                submodule (archie2x/tuxmath   @ a2x/sdl3_port)
  _build/<preset>/        out-of-tree build directory (gitignored)
  _build/<preset>/stage/  install destination for `cmake --install`
----
